# **Programming challenge GrowthHackers**

Desenvolvimento de uma API de autenticação.

### Tecnologias

- NodeJS
- Express
- Knex (ORM)
- Tsyringe (Injeção de dependências com decorators)
- Jest (Testes)
- Eslint
- Prettier

## Rodando as migrations

```
npm run db:migrate
```

ou

```
yarn db:migrate
```

## Rodando o projeto

```
npm run dev:server
```

ou

```
yarn dev:server
```

## Rodando os testes

```
npm run test
```

ou

```
yarn test
```
